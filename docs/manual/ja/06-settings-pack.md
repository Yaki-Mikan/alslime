# 06 設定のインポート / エクスポート

作り込んだ設定一式を zip の「設定パック」として書き出し・取り込みできます。バックアップ、別 PC への引っ越し、設定の配布・共有に使えます。

開き方: 設定（歯車）→「設定パック」。「エクスポート」と「インポート」の2タブがあります。

## 1. エクスポート（書き出し）

![設定パックのエクスポート](../images/ja/06-01-settings-pack-export.png)

1. 書き出す設定の種類を選びます。種類はグループごとに並んでいます（初期状態はすべて選択）。
   - **ロールプレイ設定**: キャラクター・シチュエーション・世界観・舞台など
   - **項目設定・プリセット**: 項目設定、各種プリセット
   - **各種設定**: 置換設定・心情定義・モデル一覧（ユーザー編集）など
   - **画像生成設定**: タグ判定指示・プロファイルなど（支援者機能が有効な場合のみ）
2. キャラクターを含める場合は「キャラクター画像を含める」も選べます（サイズが大きくなります）。
3. 「パック名（任意）」を付けて「エクスポート（zip をダウンロード）」を押すと、ブラウザに zip がダウンロードされます。

> **パックに入らないもの**: 認証情報・会話履歴・キャッシュ・ログ、およびポートや接続先などの環境依存設定は、安全のため常に除外されます。

## 2. インポート（取り込み)

![設定パックのインポート](../images/ja/06-02-settings-pack-import.png)

1. 「パックファイル（zip）を選択」で設定パックを選びます。
2. 自動で中身が検査され、「取り込みプラン」が表示されます。各ファイルが「新規」「衝突」「スキップ」に色分けされます。
3. 既存ファイルと衝突がある場合は、扱いを選びます。
   - **スキップ（既存を残す）**（既定）／**上書き**／**リネームして追加**
   - 一括設定のほか、衝突したファイルごとに個別指定もできます。
4. 「取り込み実行」を押すと反映され、書き込み・スキップの件数と書き込み先が表示されます。

認証情報を含むパックは取り込みできません（自動でブロックされます）。

## 3. 公式サンプルのダウンロード

インポートタブの「公式サンプル」から、公式のサンプル一式（キャラクター・テンプレート・プリセット）を GitHub からダウンロードしてそのまま取り込めます。**既存ファイルは変更しません**（新規追加のみ）。

はじめて使うとき、[03章](03-character.md) のキャラクター選択に何も出てこない場合は、まずここを押してサンプルを入れるのが近道です。

## 4. 起動時の自動取り込み（import_inbox）

画面操作なしで取り込む方法もあります。起動フォルダの `roleplay/import_inbox/` にパック zip を置いて AlSlime を起動すると、自動で取り込まれます。

- **新規ファイルのみ**取り込みます。既存ファイルは変更しません。
- 処理した zip は `roleplay/import_inbox/processed/` へ移動されます（30日後に自動削除）。
- 1回の起動で処理するのは10件まで。超えた分は次回起動時に処理されます。
- 今回の起動での取り込み結果は、設定パックのインポートタブ下部「起動時取り込み（import_inbox）」で確認できます。
