# 07 支援者機能

AlSlime は無料で使えます。開発を GitHub Sponsors で支援していただいた方には、追加機能（画像生成の ComfyUI 連携など）が開放されます。

## 1. 支援者機能の画面

開き方: 設定（歯車）→「支援者機能」。

![支援者機能](../images/ja/07-01-sponsor-modal.png)

上部に現在の支援状態がバッジで表示されます。

| 表示 | 意味 |
| --- | --- |
| 未認証（free） | ログインしていない状態。無料機能のみ |
| 有効 | 支援が確認済みで、支援者機能が使える状態 |
| 更新待ち（猶予期間中） | 支援状態の再確認待ち。**猶予期間中も機能はそのまま使えます** |
| 失効 | 有効期限切れ。再ログインまたは支援の再開が必要 |
| 無効なトークン | 認証情報に問題がある状態。再ログインしてください |

バッジの横には支援プラン名と有効期限が表示されます。

## 2. ログインする

1. 「GitHub でログイン」を押すと、ブラウザで GitHub の認可ページが開きます。
2. GitHub Sponsors で支援中のアカウントで認可すると、アプリ側に自動で反映されます（画面に「ブラウザでログインを完了してください...」と表示されている間に、ブラウザ側の操作を済ませてください）。
3. ページが自動で開かない場合は「開かない場合はこちら」のリンクから開けます。

ログイン後は「状態を更新」で支援状態の再確認、「ログアウト」で認証の解除ができます。認証状態は普段は自動で維持されるため、通常はログインし直す必要はありません。

## 3. モジュールを受け取る

支援が有効になると、追加機能の本体（モジュール）をダウンロードできます。

![モジュールのダウンロード](../images/ja/07-02-sponsor-modules.png)

- **ComfyUI 連携モジュール（画像生成）**: 会話からの画像生成機能（[08 ComfyUI 連携](08-comfyui.md)）
- **行動選択肢モジュール**: 応答に行動の選択肢を提示する機能

「モジュールをダウンロード」を押すと取得・配置され、「配置済み（再起動後に有効になります）」と表示されます。**AlSlime を再起動すると有効**になり、表示が「有効（サイドカー動作中）」に変わります。

行動選択肢モジュールは、配置後に表示される「行動選択肢を有効にする」で、再起動なしに機能の ON / OFF を切り替えられます。

## 4. うまくいかないとき

- 「有効な支援が見つかりませんでした」: ログインしたアカウントが GitHub Sponsors で支援中か確認してください。
- 「トークンの検証に失敗しました」: アプリを最新版へ更新してから再試行してください。
- モジュールの取得に失敗する: ネットワーク接続を確認して再試行してください。
- そのほかは [09 困ったときは](09-troubleshooting.md) へ。
