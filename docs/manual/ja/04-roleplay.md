# 04 ロールプレイ設定

キャラクター以外のロールプレイ設定——世界観・舞台・シチュエーション、日付時刻、プリセット、置換やパラメータ項目などの高度な設定——を説明します。

キャラクターの選択と会話開始の基本は [03 キャラクターと会話する](03-character.md) を先にどうぞ。

## この章でやること

1. 環境（世界観・舞台・シチュエーション・ユーザー）を設定する
2. 文体と全体の追加設定を使う
3. 日付時刻設定で会話内の時間を操る
4. プリセットで設定一式を保存・呼び出しする
5. 高度な設定（テキスト置換・パラメータ項目・テンプレート）

## 1. 環境を設定する

会話設定サイドバー（ヘッダー右端の三本線）の「環境」セクションを開くと、上から「世界観欄」「舞台欄」「シチュエーション欄」「ユーザーの設定欄」が並んでいます。

![環境セクション](../images/ja/04-01-environment.png)

- 各欄はプルダウンで選びます。1つ選ぶと空きの枠が増え、**各欄とも最大5件**まで重ねられます。枠のゴミ箱ボタンで外せます。
- 各欄の下の「追加◯◯設定」トグルを ON にすると自由記述欄が現れ、選んだ設定ファイルに書き足す形で内容を補えます。ファイルを作るほどではない、その場かぎりの設定に便利です。

それぞれの選択肢は、起動フォルダの次の場所にある Markdown ファイルの一覧です（サブフォルダで整理すると「フォルダ名/ファイル名」として表示されます）。

| 欄 | フォルダ |
| --- | --- |
| 世界観欄 | `roleplay/global/worldviews/` |
| 舞台欄 | `roleplay/global/stages/` |
| シチュエーション欄 | `roleplay/global/situations/` |
| ユーザーの設定欄 | `roleplay/users/` |

## 2. 文体と全体の追加設定

- **文体設定欄**: 応答の文体（`roleplay/global/writing_styles/` のファイル）を選びます。「追加文体設定」で自由記述の補足もできます。
- **画像生成補足事項**: 画像生成機能（[08 ComfyUI 連携](08-comfyui.md)）を使う場合に、生成時の分析へ渡す補足を書いておく欄です。
- **全体の追加設定**: どのカテゴリにも属さない自由記述の設定枠です。トグルを ON にして書き込むと、会話全体への指示として扱われます。

## 3. 日付時刻設定

会話の中の「いま何時か」を管理できます。サイドバーの「日付時刻設定」セクションで設定し、有効にすると見出しに「有効」バッジが付きます。

![日付時刻設定](../images/ja/04-02-datetime.png)

- **日付時刻をプロンプトに含める**: このトグルが大元のスイッチです。横の鍵アイコンでロックしておくと、次回以降も有効状態で始まります。
- **現在日付時刻 / 固定日付時刻**: 実際の現在時刻を使うか、指定した日時（年・月・日・時・分）で固定するかを選びます。固定側には「日付を今日にする」の切り替えもあります。
- **時刻インクリメント**: 送信ごとに、設定した時間が経過したものとして会話内の時刻を進めます。「詳細モード（年月日秒）」でより細かく刻めて、「つきあいの長さをセッション経過と連動」でキャラクターとの関係の長さも一緒に進められます。
- **セッション時刻情報**: インクリメント有効の会話では、現在のセッション時刻・累積経過日数が表示され、鉛筆アイコンで直接編集もできます。会話中は、ヘッダー左端「セッション状態」のドロワー内にも同じ情報のパネル（セッション時刻）が表示されます。

日付時刻には2種類のプリセットがあります。**日付時刻プリセット**は固定日時の値だけを、**時刻設定グループプリセット**は「日付時刻＋インクリメント設定を丸ごと」保存・復元します。

## 4. プリセットで設定一式を保存・呼び出しする

![会話設定プリセット](../images/ja/04-03-preset.png)

サイドバーには階層の違う3種類のプリセットがあります。

| プリセット | 保存されるもの | 場所 |
| --- | --- | --- |
| 会話設定プリセット | サイドバーの設定一式（キャラ・環境・文体・日時など丸ごと） | サイドバー最上部 |
| キャラ設定プリセット | キャラクターの組み合わせまわり | キャラクター欄の下 |
| 詳細設定プリセット | キャラ1人分の詳細設定 | 各キャラの詳細設定内 |

いちばんよく使うのは**会話設定プリセット**です。「新規保存」で名前を付けて保存し、「プリセットを選択...」からいつでも同じ構成で会話を始められます。お気に入りの組み合わせができたら保存しておきましょう。

## 5. 高度な設定

### テキスト置換設定

設定ファイルの読み込み時に、特定の文字列を自動で置き換えます。設定ファイルに `{{usr}}` のような変数を書いておき、実際の名前に置換する使い方を推奨します。初期状態では `{{usr}}`（→ユーザー名）と `{{char}}`（→キャラクター名）の2つのルールが登録されています。

- 開き方: 設定（歯車）→「基本チャット設定」→「テキスト置換設定」
- 各行で「置換対象」「置換後ソース（未選択（手動入力）／ユーザー名／キャラクター名）」「置換後文字列」を設定します。

![テキスト置換設定](../images/ja/04-04-replacement.png)

### キャラパラメータ項目の管理

キャラクターの内部パラメータ（[03章の状態パネル](03-character.md) に表示される項目）の構成そのものをカスタマイズできます。

- 開き方: 設定（歯車）→「基本チャット設定」→「キャラパラメータ項目の管理」
- 「項目設定管理」画面で、グループと要素（スライダー・ドロップダウン・テキスト・トグルなど）を組み合わせて項目設定ファイルを作ります。
- 作った項目設定は、会話設定サイドバーの「**パラメータ項目設定**」プルダウンで会話に適用します（セッション開始後は変更できません）。

![項目設定管理](../images/ja/04-05-schema-editor.png)

### テンプレートの管理

設定ファイルエディタで新規作成するときの雛形（テンプレート）を編集できます。

- 開き方: 設定ファイルエディタ（ヘッダーのノートアイコン）→「テンプレートを管理」
- 種別ごとにテンプレートを作成・編集し、「デフォルトに設定」したものが新規作成時の初期内容になります。

![テンプレートエディタ](../images/ja/04-06-template-editor.png)
