# 08 ComfyUI 連携（画像生成・支援者向け）

会話の内容から ComfyUI で画像を生成する、支援者向け機能です。

## 前提条件

- **支援者機能が有効**であること（[07 支援者機能](07-sponsor.md) でログインし、ComfyUI 連携モジュールを配置・再起動済み）
- **ComfyUI 本体**がご自身の環境に導入済みで、起動していること（導入は ComfyUI の公式手順に従ってください）

## 1. ComfyUI と接続する

開き方: 設定（歯車）→「画像生成設定」（支援者機能が有効な場合のみ表示されます）。

![ComfyUI 画像生成設定](../images/ja/08-01-comfyui-settings.png)

1. 「接続設定」に ComfyUI の URL を入力します（既定: `http://127.0.0.1:8188`。ComfyUI のデフォルトポートは 8188 です）。
2. 「テスト」を押して「接続できました。」と出れば接続は完了です。

## 2. 生成が動くまでの準備

接続のほかに、次の3つが揃うと生成できるようになります。

1. **ワークフロー**: 「ワークフロー設定」に、ComfyUI の「Save (API Format)」で保存した JSON ファイルをドラッグ&ドロップで登録し、使用テンプレートとして選択します（UI フォーマットの JSON は登録できません）。
2. **タグ判定 AI**: 「タグ判定プロンプト設定」で、会話から画像生成タグを判定する AI（Gemini / Claude / Antigravity）とモデルを選びます。
3. **生成プロファイルとタグ判定指示ファイル**: この2つは**セットで機能**します。自分で用意するほか、[06章の公式サンプル](06-settings-pack.md) に含まれるものを取り込むのが近道です。

## 3. キャラクター別の画像生成設定

キャラクターの見た目をタグや LoRA として登録しておくと、そのキャラクターらしい画像が生成されます。

- 開き方: 会話設定サイドバーのキャラクター枠にある画像アイコン（このキャラクターの画像生成設定を開く）、または画像生成設定内の「キャラクター画像生成設定」。
- 主な設定: キャラクター名・作品名、Danbooru タグ検索、キャラクタープロンプト、身体的特徴、LoRA（強度・トリガーワード）、服装設定（AI が服装名を返したとき対応するプロンプトと LoRA を適用）、追加ポジティブ／ネガティブ。

画面幅の広い環境では、これらをひとつの画面で編集できる**統合設定**が開きます（設定ファイルエディタの「画像生成統合設定」タブからも開けます）。左側で設定を編集しながら、右側のテスト生成で仕上がりを確かめられる構成です。

![画像生成統合設定](../images/ja/08-02-integrated-settings.png)

## 4. 会話から画像を生成する

1. 会話中、AI の応答メッセージに「**画像生成**」ボタンが表示されます。
2. 押すと、タグ判定 AI が会話の流れから場面を読み取り、ComfyUI で生成が始まります（ボタンは「生成中...」表示になります）。
3. 生成された画像はそのメッセージに添付表示されます。クリックで拡大でき、拡大表示から「この画像を背景に設定」も選べます。

![メッセージの画像生成](../images/ja/08-03-generate-button.png)

## 5. テスト生成

会話を使わずに生成を試したいときは「画像生成テスト」を使います。テンプレート・キャラクター・タグ・プレースホルダを指定して「生成」し、結果と一緒に「解決済みプロンプト」（実際に使われたポジティブ／ネガティブ／LoRA）を確認できます。設定の調整に便利です。

![画像生成テスト](../images/ja/08-04-generate-test.png)

## 6. セッション内背景画像

画像付きの応答が表示されている間、その画像を画面の背景に表示する機能です。設定は**画像生成設定の最下部**にあります。透明度・縮尺（画面を埋める／全体を表示）・表示範囲（履歴欄のみ／チャット欄まで）を調整できます。

## うまくいかないとき

- 「接続テストに失敗しました」: ComfyUI 本体が起動しているか、URL とポートが合っているか確認してください。
- 「このファイルは UI フォーマットです」: ComfyUI で「Save (API Format)」を使って保存し直してください。
- 「タイムアウト（5分）」: ComfyUI 側の生成が重い可能性があります。ComfyUI 本体の画面で状況を確認してください。
- そのほかは [09 困ったときは](09-troubleshooting.md) へ。
